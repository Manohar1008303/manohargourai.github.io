# Loan Portfolio Optimization Analysis

A data analysis project examining an optimized leveraged-loan portfolio. It analyzes
allocation and industry exposure, visualizes risk versus return, and computes key metrics —
achieving a 3.92% expected return at a 3.27% average default probability with a portfolio
WARF of 3050 across a diversified, multi-sector book.

## Tech
Python · Pandas · Matplotlib · Seaborn · Portfolio Analytics

## Structure
- `Notebooks/portfolio_analysis.ipynb` — main analysis notebook
- `Src/data_preparation.py` — data cleaning and preparation
- `Src/portfolio_optimizer.py` — optimization and metrics
- `Data/clo_loan_dataset.csv` — loan-level input dataset (synthetic)
- `Results/` — cleaned dataset and optimized portfolio output

## Notes
Uses synthetic sample data for demonstration.
