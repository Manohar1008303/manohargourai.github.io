# E-Commerce Sales & Order Analysis

An end-to-end data visualization project analyzing a Brazilian e-commerce marketplace.
Multiple public datasets (~89K order lines) covering orders, products, customers,
payments, and sellers were joined and cleaned, then turned into six analytical Tableau
worksheets, an interactive dashboard, and a multi-step data story.

## Tech
Tableau · Data Preparation · Dashboards · Data Storytelling · Python

## Key insights
- A concentrated category mix, with toys leading order volume.
- Credit cards as the dominant payment type.
- A clear November sales spike driven by seasonal demand.
