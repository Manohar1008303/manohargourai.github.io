# Culture & Economic Performance Analysis

A statistical study examining how national culture relates to economic performance,
pairing Hofstede's six cultural dimensions with GDP per capita for 60 countries across
2020–2024. Conducted in JMP, the analysis spans correlation and hypothesis testing,
multiple linear regression with full diagnostics (VIF, residual and influence checks),
factor analysis, and GDP forecasting — translating the results into practical insights
for market-entry and investment decisions.

## Tech
JMP · Regression · Hypothesis Testing · Factor Analysis · Forecasting

## Methods
- Correlation and hypothesis testing between cultural dimensions and GDP per capita
- Multiple linear regression with diagnostics (VIF, residuals, influence)
- Factor analysis of the six Hofstede dimensions
- GDP forecasting
